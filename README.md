## ✏️ About

Tektok chat is an app to help you to connect and chat with people around the world anywhere and everywhere !


## 🔖Build With
- ReactJS
- ExpressJs
- Bootstrap

## 💻 How To Install?
- Clone This Repo
```
https://github.com/NisrinaNataraharja/FE-Tektok-Chat.git
```
- Go To Folder Repo

```
cd FE-Tektok-Chat
```

- Install Module

```
npm install
```

- Type ` npm start` To Start Website
